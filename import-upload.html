02 Tips for a smooth import
FORMAT
Use a header row
First row should contain column names like "Full Name", "Email", "Company". The AI mapper uses these to suggest target fields.
ENCODING
UTF-8 preferred
Special characters in names ("Schönberger", "李") only render correctly with UTF-8. Other encodings will be auto-detected but may flag rows.
SIZE
100k rows / 100MB
Larger files? Split into batches or use the API. Imports run in the background; you'll get an email when done.
